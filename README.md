Phishing Website Detection using Machine Learning

This project is a machine learning-based web application designed to detect phishing websites. 
The system uses trained classification models to analyze website-related features and predict whether a website is legitimate or phishing.

The project was developed using Python and implemented with a Flask web application to provide an interactive interface for users.

Key Features
- Machine learning models trained on phishing website dataset
- Random Forest and XGBoost algorithms used for prediction
- Flask-based web interface for user input
- Real-time phishing detection

Technologies Used
- Python
- Flask
- Scikit-learn
- XGBoost
- Pandas
- NumPy

How to Run the Project
1. Clone the repository
2. Install dependencies using:
   pip install -r req.txt
3. Run the application:
   python app.py
4. Open the application in a browser at:
   http://127.0.0.1:5000
